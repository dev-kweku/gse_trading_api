API Documentation

API Overview

The GSE Trading Data API provides access to Ghana Stock Exchange trading data through a RESTful interface. All endpoints return JSON responses and use standard HTTP status codes.

Base URL

http://localhost:8000/api/v1

Response Format

All API responses are in JSON format. Successful responses return HTTP status code 200 and contain the requested data. Error responses include an error message with details about what went wrong.

Data Model

The API works with stock trading data that includes the following fields:

Field Type Description
id Integer Unique identifier
daily_date Date Trading date
share_code String Stock symbol
year_high Float 52-week high price
year_low Float 52-week low price
opening_price Float Opening price
closing_price Float Closing price
price_change Float Price change from previous close
total_shares_traded Integer Total shares traded
total_value_traded Float Total value traded
scraped_date DateTime Date and time when data was scraped

API Endpoints

Get Stocks

Retrieve stock data with optional filtering

GET /stocks
GET
Parameters
  • skip (integer, optional): Number of records to skip (default: 0)
  • limit (integer, optional): Maximum number of records to return (default: 100)
  • share_code (string, optional): Filter by specific share code
  • start_date (date, optional): Filter by start date (inclusive)
  • end_date (date, optional): Filter by end date (inclusive)

fetch('http://localhost:8000/api/v1/stocks?limit=10&share_code=MTNGH')

Authentication

Currently, the GSE Trading Data API does not require authentication for accessing the data. However, we plan to implement API key-based authentication in future versions to enhance security and monitor usage.

When authentication is implemented, you'll need to include your API key in the request headers using the X-API-Key header field.

Future Authentication Example

fetch('http://localhost:8000/api/v1/stocks', {
  headers: {
    'X-API-Key': 'your_api_key_here'
  }
})
.then(response => response.json())
.then(data => console.log(data));

Error Handling

The GSE Trading Data API uses standard HTTP status codes to indicate the success or failure of API requests. In case of errors, the API returns a JSON object with details about the error.

HTTP Status Codes

Status Code Meaning
200 OK Request successful
201 Created Resource successfully created
400 Bad Request Invalid request parameters
404 Not Found Requested resource not found
422 Unprocessable Entity Validation error
500 Internal Server Error Server error

Error Response Format

In case of an error, the API returns a JSON object with the following structure:

{
  "detail": "Error message describing what went wrong"
}

Example error response for a validation error:

{
  "detail": [
    {
      "loc": ["body", "daily_date"],
      "msg": "field required",
      "type": "value_error.missing"
    }
  ]
}

Python Example

import requests
import json

# Base URL for the API
BASE_URL = "http://localhost:8000/api/v1"

# Function to get all stocks
def get_all_stocks():
    try:
        response = requests.get(f"{BASE_URL}/stocks?limit=100")
        response.raise_for_status()  # Raise an exception for HTTP errors
        data = response.json()
        print(json.dumps(data, indent=2))
        return data
    except requests.exceptions.RequestException as e:
        print(f"Error fetching stocks: {e}")
        return None

Data Visualization Example

// Using Chart.js to visualize stock price trends
import { Chart, registerables } from 'chart.js';
Chart.register(...registerables);

async function visualizeStockHistory(shareCode) {
  try {
    // Fetch stock history data
    const response = await fetch(`http://localhost:8000/api/v1/stocks/${shareCode}/history`);
    const data = await response.json();
    
    // Extract dates and prices
    const dates = data.history.map(item => item.daily_date);
    const closingPrices = data.history.map(item => item.closing_price);
    
    // Create chart
    const ctx = document.getElementById('stockChart').getContext('2d');
    new Chart(ctx, {
      type: 'line',
      data: {
        labels: dates,
        datasets: [{
          label: `${shareCode} Closing Price`,
          data: closingPrices,
          borderColor: 'rgb(75, 192, 192)',
          backgroundColor: 'rgba(75, 192, 192, 0.2)',
          tension: 0.1
        }]
      },
      options: {
        responsive: true,
        plugins: {
          title: {
            display: true,
            text: `${shareCode} Stock Price History`
          }
        },
        scales: {
          y: {
            beginAtZero: false,
            title: {
              display: true,
              text: 'Price (GH¢)'
            }
          },
          x: {
            title: {
              display: true,
              text: 'Date'
            }
          }
        }
      }
    });
  } catch (error) {
    console.error('Error visualizing stock history:', error);
  }
}

// Usage
visualizeStockHistory('MTNGH');
